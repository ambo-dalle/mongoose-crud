## REST API

#### Books
| Route        | Http           | Descriptin  |
| ------------- |:-------------:| -----:|
| /books      | GET   | Get all the books |
| /books/:id      | GET   | Get all one books |
| /books      | POST   | Create / Add a Book |
| /books:id     | DELETE      |  Delete a Book|
| /books | PUT      |    Update a Book |
